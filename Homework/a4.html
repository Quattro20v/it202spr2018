Answers to questions
  1. A box model refers to the design and layout of a website. In other words its a box that wraps around other elements, such as: margins, borders, padding, and the actual content
  2. A block-level element, is an element that always starts on a new line and takes up the full width available.
  3. Div elements are by default rendered as block-level elements
  4. An inline element, is an element that does not start on a new line and only takes up as much width as necessary.
  5. An example of an inline element would be: br
  6. CSS should be included in pages either internally or externally, depending on the website
  7. Selectors are used to select the element or elements that need to be styled.
  8. CSS rule to set the backround color black of the body element to: #0000FF is: body { background-color: #0000FF; }
