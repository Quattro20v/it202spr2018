Question # Answer
1 Bootstrap can be either downloaded from getbootstrap.com or included from a CDN.
2 CDN is a Content Delivery Network, that does not require us to download bootstrap. This way it can just be included in the webpage.
3 An advantage to using a CDN is a faster loading time. The users usually have bootstrap dowloaded in their cache from the specific CDN. Also a CDN allows the user to download bootstrap, from a server closest to them.
4 In responsive design break points are points at which the design changes, depending to the size of the screen. Or zooming in and out.
5 The new break points are at 576px, 768px, 992px and 1200px
6 The fixed width container, is a fixed width container that is responsive. While the width container is also a responsive container, it expands accross all of the width of the device.
7 Bootstrap breaks the page into 12 columns
8 Rows put the contents on the same row, to make sure that they are their even when the screen size changes, making them smaller etc..
